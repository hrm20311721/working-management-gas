# 稼働エンジニア メール一括送信システム（GAS）

Googleスプレッドシートで管理している稼働エンジニアに対して、作業報告書・請求書の提出依頼メールを一括送信するGoogle Apps Scriptシステムです。

![Google Apps Script](https://img.shields.io/badge/Google%20Apps%20Script-GAS-blue?logo=google)
![Gmail](https://img.shields.io/badge/Gmail-API-red?logo=gmail)
![Google Sheets](https://img.shields.io/badge/Google%20Sheets-連携-green?logo=googlesheets)

---

## システム概要

毎月の作業報告書・請求書提出依頼メールの送信業務を自動化するシステムです。スプレッドシートのサイドバーから対象月とシートを選択するだけで、送信対象者を自動抽出してメールの下書きを一括作成します。

---

## 機能一覧

| 機能 | 説明 |
|---|---|
| サイドバー表示 | スプレッドシートにカスタムメニューを追加し、操作用サイドバーを表示 |
| シート選択 | 月次シートを選択して対象エンジニアのリストを取得 |
| 送信対象の自動抽出 | メールアドレスの形式チェック・送信済みフラグで対象を自動絞り込み |
| メール一括送信 | 各エンジニアに個別の作業報告書・請求書締切日を記載したメールを送信 |
| 送信済みフラグ | 送信後にスプレッドシートへ「済」を自動記録 |

## ファイル構成

```
working-management/
├── main.gs     メイン処理（シート読み込み・メール送信）
└── index.html  サイドバーUI（月・シート選択・送信ボタン）
```

---

## スプレッドシートの構成

4行目以降に以下の形式でデータを入力してください：

| 列 | 内容 |
|---|---|
| D列 | 送信済みフラグ（「済」で送信対象外） |
| E列 | 企業名 |
| F列 | エンジニア名 |
| G列 | メールアドレス |
| I列 | 作業報告書の提出期限 |
| J列 | 請求書の提出期限 |

## セットアップ手順

**1. スプレッドシートを準備**
- Googleスプレッドシートを新規作成
- スクリプトエディタを開き、`main.gs` と `index.html` をコピー

**2. スプレッドシートIDを設定**
- `main.gs` の1行目にスプレッドシートのIDを設定
```javascript
const ss = SpreadsheetApp.openById('YOUR_SPREADSHEET_ID');
```

**3. 送信者情報を設定**
- `main.gs` 内の以下のプレースホルダーを実際の情報に置き換える
```
YOUR_COMPANY_NAME  → 会社名
YOUR_NAME          → 担当者名
your-email@example.com → メールアドレス
cc1@example.com    → CCアドレス
```

**4. トリガーを設定**
- スクリプトエディタの「トリガー」からスプレッドシートオープン時に `addMenu` を実行するよう設定

**5. 動作確認**
- スプレッドシートを開きカスタムメニュー「稼働管理システム」→「サイドバー表示」
- 月とシートを選択して送信対象者を確認
- ※ `main.gs` の `draft.send()` はデフォルトでコメントアウトされています。動作確認後にコメントを外してください

---

## 使い方

1. スプレッドシートのメニューから「稼働管理システム」→「サイドバー表示」
2. 対象月をプルダウンで選択
3. 当月のシートをプルダウンで選択 → 送信対象者が自動表示
4. 内容を確認して「送信」ボタンをクリック
5. 送信完了後、スプレッドシートのD列に「済」が自動記録される

---

## 工夫した点

- **メールアドレスの正規表現バリデーション**：不正なアドレスへの誤送信を防止
- **送信済みフラグによる重複防止**：「済」の行は自動的に送信対象から除外
- **下書き作成方式**：`createDraft()` で下書きを作成してから送信するため、送信前の内容確認が可能
- **日付の自動フォーマット**：スプレッドシートのDate型を「M月d日」形式に変換して本文に挿入
